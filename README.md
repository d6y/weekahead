# Generates a week-ahead markdown file for use in Bear

## Install guide

```
cargo install --path .
```

## Usage

```
$ weekahead
```

...will output a markdown file dated for Monday.

## Example

```

$ cat Week\ 19\ Oct\ 2020.md
# Week 19 Oct 2020
## 19 Oct 2020 (Mon)


#Journal/OnThisDay/10/19# #Journal/2020/10#

---

## 20 Oct 2020 (Tue)


#Journal/OnThisDay/10/20# #Journal/2020/10#

---

```

...and so on until Sunday.
